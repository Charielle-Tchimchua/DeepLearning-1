# DeepLearning-4: Advanced Vision and 3D Data

## Description
Ce projet implémente un **U-Net 2D** pour la segmentation d'images médicales et un **bloc Conv3D** pour les données volumétriques, avec suivi MLOps via MLflow.

## Prérequis
- Python 3.8+
- TensorFlow 2.10+
- MLflow
